XEA Tools

Otomatisasi pengunduhan, penggabungan, dan pelaporan dokumen PDF (STBA & STAT) untuk teknisi via Termux Android.

🚀 Cara Instalasi di Android

Aplikasi ini dirancang untuk berjalan secara lokal di perangkat Android menggunakan Termux. Untuk menginstal seluruh kebutuhan (Python, Flask, PyPDF) dan mengonfigurasi otomatis, cukup jalankan perintah berikut di Termux Anda:

curl -sSL https://raw.githubusercontent.com/ShadowSoldiers/xeatools/main/install.sh | bash

Catatan: Installer akan meminta Anda untuk memasukkan kredensial Galva XEA dan opsi konfigurasi email.

✨ Fitur Utama

📥 Auto Download

Tarik dokumen dari API Galva XEA berdasarkan rentang tanggal. Otomatis mengonversi dokumen format JPG menjadi PDF (jika ada).

🔗 Smart Merge

Otomatis mencocokkan dan menggabungkan dokumen STBA dan STAT berdasarkan nomor order. Menghindari pemrosesan duplikat.

📧 Email & Laporan

Kirim hasil penggabungan via email langsung ke klien/manajemen, lengkap dengan rincian nama pelanggan dan nomor seri.

🌐 Local Web GUI

Memiliki antarmuka pengguna berbasis web (Flask) yang cantik dan responsif, dapat diakses langsung dari Chrome Android via localhost.

🛠 Cara Menjalankan Aplikasi

Setelah instalasi selesai, Anda bisa menjalankan aplikasi melalui *shortcut* yang ada di Termux:Widget, atau jalankan perintah manual berikut:

cd ~/merge_pdf bash start_manual.sh

Lalu buka peramban web (Chrome) di HP Anda ke tautan: http://localhost:5000
